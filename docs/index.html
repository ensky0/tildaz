Fast, focused, native to Windows and macOS

Built around the things a drop-down terminal has to get right.

⚡

F1 global hotkey

Drop down from any app with one key. Press again to hide. Alt+Enter toggles monitor fullscreen.

📑

Tabs, drag, rename

Ctrl+Shift+T to open, Alt+1–9 to switch, drag to reorder, double-click the tab to rename.

🎨

GPU rendering

Direct3D 11 + ClearType subpixel on Windows. Metal + retina glyph atlas on macOS. DirectWrite / CoreText glyph atlases with HLSL / MSL shaders.

🚀

Native PTY on both

Bundled OpenConsole ConPTY (1.26× faster than system conhost) on Windows. POSIX openpty + login_tty + IUTF8 termios on macOS.

🖥️

Multi-monitor, DPI-aware

Drops onto whichever monitor the cursor is on. Reacts instantly to resolution, DPI, and taskbar changes.

🌏

Unicode, CJK, emoji, IME

Correct wide-character rendering, color emoji including ZWJ family glyphs, and inline IME pre-edit (Korean / Japanese / Chinese). Font fallback chain up to 8 deep.

🎭

18 built-in themes

Ghostty, Windows Terminal, Dracula, Catppuccin, Tokyo Night and more. COLORFGBG is set automatically so vim picks the right scheme.

🪟

Translucent, always on top

Docks to any screen edge (top/bottom/left/right). Size and offset are specified as percentages of the monitor.

📋

Select to copy

Drag to select, release to copy. Double-click for word, middle-click to paste.

Quick start

1

Download

From the Releases page: tildaz-vX.Y.Z-windows.zip (Windows) or tildaz-vX.Y.Z-macos.dmg (macOS, universal Apple Silicon + Intel).

Releases →
2

Install

Windows: unzip anywhere. Keep tildaz.exe, conpty.dll, and OpenConsole.exe together — the bundled ConPTY runtime activates only when all three sit in the same folder.

macOS: open the DMG, drag TildaZ.app into the Applications shortcut. macOS 13 Ventura or later required.

3

Run and hit F1

First launch writes the config with defaults: %APPDATA%\tildaz\config.json (Windows) or ~/.config/tildaz/config.json (macOS). Shell, theme, font, and drop-down geometry live there.

macOS first launch: the app is ad-hoc signed; Gatekeeper may block the first open. Right-click → Open, or run xattr -d com.apple.quarantine /Applications/TildaZ.app. Then grant Input Monitoring + Accessibility once for the global hotkey.

Heads up on EDR false positives (Windows): the current tildaz.exe is not yet Authenticode-signed. Some endpoint security products (SentinelOne, CrowdStrike, etc.) flag unsigned ConPTY-spawning binaries by behavior alone. Code signing via the SignPath Foundation OSS program is in progress.

Build it yourself

Install Zig 0.15.2 and you're one command away.

# default — ReleaseFast
zig build

# debug
zig build -Doptimize=Debug

# release zip + SHA256
zig build package
