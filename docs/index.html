Why TildaZ

The drop-down workflow people love, rebuilt for the desktops they actually use.

Instant

One key between thought and shell

Drop the terminal over your current workspace, run the command, and hide it again before your flow cools off.

Linux

Wayland coverage that matters

Everywhere

Same reflex, every OS

Keep one JSON config shape and one muscle memory across Linux, macOS, and Windows while each host still behaves natively.

Native

Unicode

Linux

Every major Wayland desktop

A direct Wayland client with no GTK or Qt runtime dependency. Real drop-down placement and global hotkey paths across major desktops, with package formats for common Linux distribution families.

  • KDE Plasma, GNOME, Cinnamon, COSMIC, Hyprland, sway
  • Fractional / HiDPI scaling, fcitx5 / IME support
  • tar.gz · deb · rpm · AppImage · Arch pkg (x86_64 / aarch64)

macOS

Native, Retina-sharp

Built on AppKit and Metal with full IME composition, candidate anchoring, and Hanja / kanji reconversion. Universal Apple Silicon + Intel.

  • Retina glyph rendering
  • Universal app bundle
  • Cursor-following drop-down on any display

Windows

Fast, native, bundled

GPU-accelerated rendering with a bundled Microsoft OpenConsole runtime for a consistent, modern terminal — x64 and ARM64.

  • Crisp ClearType glyph rendering
  • Bundled ConPTY for reliable shells
  • Win32 global hotkey and native window behavior

Get Started

Download, install, press F1.

1

Download

Grab the latest GitHub Release for your OS: a Linux package (deb / rpm / AppImage / Arch / tar.gz), macOS DMG, or Windows zip.

2

Install

Linux: install the package or run ./install.sh from the tarball. macOS: drag the app into Applications. Windows: unzip and run.

3

Press F1

Drop the terminal down with the hotkey. Tweak config.json for shell, theme, font, hotkey, opacity, and geometry.
