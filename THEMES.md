## Vim / TUI dark/light auto-detection

`COLORFGBG` is set automatically based on the theme background luminance, so vim's `:set background?` picks the right scheme without manual configuration. tmux and less use the same convention. On Windows the value is propagated into WSL via `WSLENV`.
